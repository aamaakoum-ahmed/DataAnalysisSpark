# Analyse S&P 500 avec Spark
Pour lancer ce projet :
1. Telechargez le dataset `sp500_stock_price.csv` (564 Mo).
2. Placez-le dans HDFS a l'adresse `/data/sp500/`.
3. Lancez le script dans le spark-shell avec `:load analysis.scala`.
